Analyse des ventes e-commerce – Python & Power BI
Présentation du projet

Ce projet vise à analyser un dataset e-commerce afin de comprendre les performances commerciales, les tendances de vente et le comportement des clients.

L’analyse combine Python pour la préparation et l’exploration des données et Power BI pour la création d’un tableau de bord interactif permettant de visualiser les indicateurs clés de performance (KPI).

L’objectif est de transformer des données transactionnelles brutes en insights business exploitables pour aider à la prise de décision.

Objectifs de l’analyse

Les principaux objectifs du projet sont :

analyser la performance globale des ventes

identifier les catégories de produits les plus performantes

comprendre le comportement d’achat des clients

mesurer les indicateurs clés de performance (KPI)

analyser la distribution géographique des ventes

identifier les opportunités d’amélioration pour l’entreprise

Dataset

Le dataset utilisé dans ce projet provient du Brazilian E-commerce Public Dataset by Olist.

Il contient des données transactionnelles d’une plateforme e-commerce incluant :

identifiant de commande

identifiant client

ville et état du client

catégorie de produit

type de paiement

date de commande

valeur de la commande

informations de livraison

Le dataset complet est disponible ici :

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

En raison de la taille des fichiers, le dataset brut n’est pas inclus dans ce repository.
